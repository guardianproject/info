**A Developer Guide to Integrating Tor into Mobile Applications**

Onion Mobile Devs is a comprehensive guide for integrating [Tor](https://torproject.org) into mobile applications to enhance user privacy and security. Tor is free and open-source software for enabling anonymous communication, masking user location and online activity through a global network of volunteer relays.

By following this guide, developers gain knowledge about the various methods to integrate Tor into mobile platforms while following open-source development best practices. The guide supports the broader objective of advancing digital privacy rights and internet freedom globally.

The guide also covers [Orbot](https://orbot.app), the original Tor on Mobile app, demonstrating the practical application of these concepts.

**Read the full guide at [guide.onionmobile.dev](https://guide.onionmobile.dev/)**
