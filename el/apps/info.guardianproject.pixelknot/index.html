PixelKnot: Hidden Messages

Έχετε ένα μυστικό που θέλετε να μοιραστείτε; Γιατί να μην το κρύψει σε μια εικόν

GitHub - Blog Posts - Μεταφράση - GPL-3.0-only
















Έχετε κάποιο μυστικό που θέλετε να μοιραστείτε; Γιατί δεν το κρύβετε σε μια εικόνα; Με το PixelKnot μόνο οι φίλοι σας με το μυστικό κωδικό μπορούν να ξεκλειδώσουν το ειδικό μήνυμά σας. Όλοι οι άλλοι βλέπουν απλά μια ωραία εικόνα. Έχει πλάκα και είναι ένας εύκολος τρόπος για να μοιράζεστε κρυφά μηνύματα, χωρίς να το γνωρίζει κανένας. Πάρτε τα πίξελ, κρύψτε εκει μέσα το μήνυμά σας και δείτε και εσείς!

Αυτή η εφαρμογή στεγανογραφίας περιλαμβάνει:

  • ΜΕΤΑΜΦΙΕΣΤΕ ΤΑ ΜΗΝΥΜΑΤΑ ΣΑΣ: Οι εικόνες είναι δημόσιες. Το μήνυμα που σκοπεύετε να κρύψετε μέσα, οχι. Ακόμα και κάποιος με εξασκηση στον εντοπισμό κάποιας επεξεργασίας στο βιογραφικό του δεν θα καταλάβει τίποτα [a]
  • ΜΟΝΟ ΓΙΑ ΕΣΑΣ: Υπάρχει δυνατότητα για την κρυπτογράφηση του μηνύματος, ωστε το μήνυμα να φτάσει μόνο εκεί που θέλετε να φτάσει.
  • ΑΝΘΕΚΤΙΚΟ ΣΕ ΕΠΙΘΕΣΕΙΣ: Δοκιμάσαμε στεγανογραφημένες εικόνες που δημιουργήθηκαν με το Pixelknot χρησιμοποιόντας μια ειδική έκδοση του Stegdetect, ενός αυτόματου προγράμματος που ανιχνεύει στεγανογραφήματα σε εικόνες.
  • ΜΙΛΑΜΕ ΤΗΝ ΓΛΩΣΣΑ ΣΑΣ: Ή τουλάχιστον προσπαθούμε. Δεν βλέπετε την γλώσσα σας; Ελάτε μαζί μας και βοηθήστε να μεταφραστεί η εφαρμογή: https://hosted.weblate.org/projects/guardianproject

Η καινούρια έκδοση του Pixelknot έχει βελτιώθεί στους εξείς τομείς:

  • ΕΥΚΟΛΟΤΕΡΗ ΚΟΙΝΟΠΟΙΗΣΗ: Είναι πλέον πολύ πιο εύκολο να κοινοποιήσετε μία εικόνα που επεξεργαστήκατε με το Pixelknot με τις εφαρμογες που χρησιμοποιείτε τυπικά για να επικοινωνήσετε με φίλους. Θυμηθείτε πως δεν επιτρέπουν όλα τα προγράμματα να στείλετε την εικόνα αυτούσια, χωρίς καμία επεξεργασία. Έτσι, όταν θέλετε να κοινοποιήσετε κάποια εικόνα, το Pixelknot θα σας προτείνει, χωρις να σας εμποδίσει να χρησιμοποιήσετε κάτι άλλο, τις εφαρμογές που είμαστε σίγουροι πως θα δουλέψουν. Βέβαια, προτάσεις άλλων εφαρμογών είναι πάντοτε δεκτές.
  • ΑΠΛΟΠΟΙΗΜΕΝΗ ΑΠΟΣΤΟΛΗ / ΛΗΨΗ: Θέλετε να στείλετε ένα μήνυμα; Ανοίξτε το Pixelknot άμεσα και η εφαρμογή θα σας δείξει ακριβώς τι να κάνετε. Λάβατε μία στεγανογραφημένη εικόνα; Πατήστε κοινοποίηση στην εφαρμογή με την οποία τη λάβατε ή στη συλλογή και επιλέξτε το Pixelknot - Θα αποκωδικοποιήσει την εικόνα αυτόματα.
  • ΝΕΟΣ ΕΠΙΛΟΓΕΑΣ ΕΙΚΟΝΑΣ: Μπορείτε να χρησιμοποιήσετε την κάμερα για να βγάλετε εσείς την φωτογραφία που θα στεγανογρφήσετε ή να επιλέξετε από την συλλογή εικόνες που υπάρχουν ήδη στη συσκευή σας. Ο νέος επιλογέας επιτρέπει την επιλογή εικόνας τόσο από τα κοινά αποθηκευτικά μεσα όσο και από ειδικές συλλογές (όπως την “ιδιωτική” συλλογή που σας προσφερει η εφαρμογή κάμερας) εάν χρειαστεί. *ΟΥΡΑ ΑΠΟΣΤΟΛΗΣ: Το Pixelknot σας επιτρέπει να δημιουργήσετε μία ουρά απο εικόνες για τη στεγανογράφηση, αντι να σας αναγκάζει να περιμένετε για την ολοκλήρωση της κρυπτογράφησης και την ταυτόχρονη αποστολή των μηνυμάτων. (Ανάλογα με το μέγεθος της εικόνας και το μήκος του κειμένου σας, αυτό θα πάρει λίγη ώρα). [b]
