Orbot: Proxy with Tor

プライバシーを強化し、ファイアウォールを突破しましょう。

GitHub - ブログ記事 - 翻訳 - BSD-3-Clause























Orbotは、他のアプリがより安全にインターネットを使用できるようにする無料のプロキシアプリです。OrbotはTorを使ってインターネットトラフィックを暗号化し、世界中の一連のコンピュータを経由してそれを隠します。Torは、個人の自由やプライバシー、機密性の高いビジネス活動や人間関係、トラフィック分析と呼ばれる国家のセキュリティを脅かすネットワーク監視の一種に対する防御を支援するフリーソフトウェアおよびオープンネットワークです。

Orbotは、本当にプライベートなインターネット接続を作成する唯一のアプリです。New York Timesが書いているように、「Torから通信が届いても、どこの誰からの通信かわかりません」。

Torは2012年に電子フロンティア財団 (EFF) パイオニア賞を受賞しました。

★ 代替品なし:OrbotはAndroidでインターネットを使用する非常に安全な方法です。その通り。Orbotは、VPNやプロキシのようにユーザーを直接接続するのではなく、世界中のコンピューターを通じて暗号化されたトラフィックを何度もバウンスします。このプロセスには少し時間がかかりますが、利用可能な最強のプライバシーとID保護は待つ価値があります。 ★ アプリのプライバシー:Orbot VPN機能を使用して、すべてのアプリをTor経由でルーティングできます。 ★ すべての人のプライバシー:Orbotは、ユーザーが使用しているアプリやアクセスしているWebサイトを、接続を監視している人に知られないようにします。ネットワークトラフィックを監視している人にわかるのは、Torを使っていることだけです。
